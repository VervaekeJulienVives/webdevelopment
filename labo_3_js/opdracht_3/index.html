Wat voor soort waarde zit er in die variabelen?

De variabele 'sliders' bevat de waarde van de balk tussen 50 en 100.

De variabele 'colordemos' bevat de kleur van het blok.

Hoeveel elementen zitten er in elke verzameling?

Elke verzameling bevat één element.

Kun je uit de HTML code afleiden waarom er precies zoveel zijn?

Ja, er is slechts één element van elke klasse aanwezig in de HTML-code.

Klik op het driehoekje naast de sliders lokale variabele zodat je het eerste element kunt zien. Op welke indexpositie in de lijst staat dit element?

Het eerste element staat op indexpositie 0 in de lijst.
