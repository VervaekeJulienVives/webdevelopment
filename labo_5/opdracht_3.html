In deze opdracht bekijken we voor 3 websites op welke manier zij hun css bijhouden

De website van vives gebruikt een link die verwezen is naar een aparte css stylesheet, ook heeft de vives website css in de head van de pagina, het gebruikt ook het id element om css toe te passen en heeft zelf ook wat inline css

De miras website gebruikt ook een link naar een apart css stylesheet, ook heeft het in de body een style blok en maakt de website ook gebruik van class en id om de css toe te passen

De vrt website gebruikt een style blok in de head van hun pagina, hiernaast gebruiken ze ook het class element om zo hun css toe te passen
